# Baluno Website

Landing page for [baluno.me](https://baluno.me) — a gentle, ADHD-friendly planner.

## Tech stack

- React 18 + TypeScript
- Vite
- Tailwind CSS + shadcn/ui
- React Router

## Development

```bash
npm install
npm run dev
```

## Deployment

Automatically deployed to GitHub Pages via GitHub Actions on every push to `main`.
